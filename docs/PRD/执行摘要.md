# 伴投 (Budvest) - 执行摘要

> **一句话定位**: 专为女性和投资小白设计的 AI 投资教育与心理陪伴产品

---

## 1. 产品概述

**伴投 (Budvest)** 是一款基于 AI 大语言模型的智能投资助手,通过 **7 位投资大师级 AI 导师** 提供个性化的投资教育和心理陪伴,帮助投资小白和女性投资者建立正确的投资观念,培养理性的投资习惯。

### 核心特点

| 特点 | 说明 |
|------|------|
| 🎓 **投教为主** | 不推荐具体股票,专注投资教育和理念培养 |
| 🤝 **心理陪伴** | 识别用户情绪,提供专业的心理疏导 |
| 💝 **女性友好** | 温暖细腻的设计,专为女性用户打造 |
| 🧠 **AI 增强** | RAG + 长期记忆,越聊越懂你 |
| 💰 **低成本** | 云原生架构,月成本 < $50 |

## 2. 市场机会

### 2.1 目标用户

**主要用户群体**: 25-40 岁女性投资者,投资经验 0-3 年

**用户规模估算**:

- 中国股民总数: ~2 亿人
- 女性占比: ~30% = 6000 万
- 投资经验 < 3 年: ~40% = 2400 万
- **目标市场规模**: 2400 万人

**初期目标**: 6 个月内获取 1 万活跃用户 (渗透率 0.04%)

### 2.2 市场痛点

1. **信息过载**: 投资小白不知如何筛选和学习投资知识
2. **情绪化决策**: 追涨杀跌,缺乏专业的心理支持
3. **女性被忽视**: 现有产品多为男性视角,缺少温度
4. **教育门槛高**: 传统投资教育晦涩难懂,成本高昂

### 2.3 竞争优势

| 维度 | 竞品 (雪球/且慢等) | 伴投 (我们) |
|------|-------------------|------------|
| **AI 能力** | ❌ 无或基础 | ✅ **Claude 3.5 深度对话** |
| **教育内容** | ⚠️ UGC 质量不稳定 | ✅ **大师级知识库 (RAG)** |
| **心理陪伴** | ❌ 缺失 | ✅ **情绪识别 + 疏导** |
| **女性友好** | ❌ 男性为主 | ✅ **专为女性设计** |
| **产品温度** | ⚠️ 工具/社区属性 | ✅ **温暖的长期伙伴** |

## 5. 技术方案

### 5.1 技术栈

```
前端: Next.js 15 + TypeScript + TailwindCSS
后端: Next.js API Routes (Serverless)
数据库: Supabase (PostgreSQL + pgvector)
AI: Anthropic Claude 3.5 Sonnet
部署: Vercel (全球 CDN)
缓存: Upstash Redis (可选)
```

### 5.3 AI 能力实现

**RAG (检索增强生成)**:

- 知识库: 100+ 大师语录 + 50+ 投资案例 + 200+ 基础概念
- 向量化: OpenAI text-embedding-3-small (1536 维)
- 存储: Supabase pgvector
- 检索: 余弦相似度,返回 top 5 相关知识

**长期记忆**:

- 工作记忆: 当前会话上下文 (最近 10 条消息)
- 短期记忆: 近 7 天对话摘要和情绪轨迹
- 长期记忆: 用户画像 (投资风格、风险偏好、决策模式)

**工具调用**:

- 基于 OpenAI Function Calling
- 7 个工具: 股价查询、K 线、资金流向、新闻、持仓风险等

## 11. 核心问题与解答

### Q1: 与现有投资社区/工具的区别?

**A**: 我们是 **AI 导师 + 教育 + 陪伴**,而非社区或工具。我们不推荐股票,专注帮助用户建立正确的投资理念和习惯。

### Q2: 如何保证 AI 回复质量?

**A**:

1. **RAG 增强**: 基于大师语录和经典案例
2. **长期记忆**: 了解用户背景,个性化回答
3. **反思机制**: 自我检查回复质量
4. **人工审核**: 定期抽查和优化 Prompt

### Q3: 如何控制 AI 成本?

**A**:

1. **缓存策略**: 高频问题缓存结果
2. **Token 限制**: 控制单次对话长度
3. **用户配额**: 免费版每日 10 次对话
4. **模型选择**: 使用 GPT-4o-mini (性价比最优)

### Q4: 合规风险如何规避?

**A**:

1. **不推荐股票**: 明确产品定位为教育,不提供具体买卖建议
2. **免责声明**: 用户协议和页面明确声明
3. **法律咨询**: 产品上线前咨询专业法律顾问
4. **用户自主**: 所有决策由用户自行判断

### Q5: 如何实现用户增长?

**A**:

1. **产品驱动 (PLG)**: 优质体验 → 口碑传播
2. **内容营销**: 小红书、知乎投放高质量内容
3. **KOL 合作**: 邀请女性理财 KOL 体验推荐
4. **裂变机制**: 邀请好友解锁高级导师
5. **社区建设**: 用户间交流,形成网络效应
